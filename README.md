# Dressly-Landing-Page

Belajar slicing desain menggunakan Bootstrap dan Sass
<br>
https://codedesign.dev/challenge/kourse

### Prerequisites

- Node.js
- npm

### Installation

```sh
git clone https://github.com/alipnf/Kourse-Hero.git
cd Kourse-Hero
npm i
```

### Usage

```sh
# run in development
npm run compile-watch

# build project
npm run compile

```
